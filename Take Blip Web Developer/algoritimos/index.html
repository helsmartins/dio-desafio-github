Algoritimos - Processo de resolução de problemas 'step by step'(passo a passo) utilizando instruções.

      Instruções:
  • o que precisa ser feito?
  • qual a ordem da execução?

Precessamento de dados - o computador recebe, manipula e armazena os dados.

Desenvolvimento do Programa

  • Análise: estudo e definição dos dados de entrada e saída
  • Algoritimo: dsecreve o problema por mio de ferrementas narrativas, fluxogrma ou pseudocódigo
  • Codificação: o algoritimo é codificado de acordo com a linguagem de programação escolhida

Algoritimo é uma seguência de passos com objetivo definido , uma execução de tarefas , um conjunto de operações que resultam em uma sucessão finitas de ações

Outra de finição de algoritimo: é uma execução de etapas para concluir uma tarefa

Etapas para construir um algoritimo

  • Compreenção do problema
  • Definição de dados de entrada
  • Definir processamento
  • Definição de dados de saída
  • Utilizar um método de construção
  • Teste e dignóstico

Formas de construir o algoritimo

  • Narrativa: utilização da linguagem natural (linguagem falada pt-br)
  • Fluxograma: utilização de símbolos pré-definidos
  • pseudocódigo: Portugol
