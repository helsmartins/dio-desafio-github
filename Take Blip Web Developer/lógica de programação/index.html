Lógica de programação

Definição: Parte da filosofia que trata das formas do ensamento e geral (dedução, indução, hipótese, inferência et.) e das operações intelectuais que visam á determinação do que é verdadeiro ou não

Lógica - auxilia a resolver um determinado problema

  • Ordenação que segue convenções
  • Organização coesa
  • Forma como desencadeiam acontecimentos
  • Forma de raciocínio

Lógica: Organização e planejamento das instruções, assertivas em um algoritimo, a fim de viabilizar a implantação de um programa.

Técnica Linear

  • Modelo tradicional(é mais simples e eficaz)
  • Não tem vínculo
  • Estrutura hierárquica
  • Programação de computadores

Técnica Estruturada

Estrutura: Organização, disposição e ordem dos elementos essenciais que compõem um corpo (concreto ou abstrato)

Objetivo: é facilitar....

  • Escrita
  • Entendimento
  • validação
  • Manutenção

Técnica Modular

Definida por módulos: controlada por um conjunto de regras.

Modelo padrão da técnica modular

  • Dados de entrada
  • Processo de transformação
  • Dados de saída

Metas:

  • Simplificação
  • Decompor o problema
  • Verificação do módulo
